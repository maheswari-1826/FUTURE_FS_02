# Client Lead Management System (Mini CRM)

## 📌 Project Description

This is a Mini CRM (Client Lead Management System) built using React.js. It helps manage client leads by allowing users to add, edit, delete, search, and update client information.

---
## 🌐 Live Demo

You can view the deployed project here:

https://future-fs-02-d3e0sodir-magi18.vercel.app

## 🚀 Features

- Add Client
- Edit Client
- Delete Client
- Search Client
- Lead Source
- Lead Status
- Notes
- Form Validation
- Local Storage
- Responsive User Interface

---

## 🛠 Technologies Used

- React.js
- HTML
- CSS
- JavaScript
- Local Storage

---

## 📥 Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```
